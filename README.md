engeld's Homebrew-Fork
========
For updating and adding formulas.

Contributing
------------
1. Fork [homebrew]
2. brew create create http://example.com/foo-0.1.tar.g
3. git checkout -b foo
4. git commit Library/Formula/foo.rb && git push
5. Pull Request
